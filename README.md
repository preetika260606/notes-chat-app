# 🧠 Jotion – Notion Clone

A full-stack Notion-style productivity app built with modern web technologies.  
Users can create, manage, and organize documents with real-time updates and dynamic UI.

---

## 🚀 Features

- 📄 Create, edit, and organize documents
- 🗂️ Nested document structure (like Notion)
- ⚡ Real-time updates using Convex
- 🔐 Authentication with Clerk
- 🖼️ Upload, change, and remove **cover images**
- ☁️ Cloud image storage with EdgeStore
- 🎨 Clean and responsive UI
- 🌙 Light/Dark mode support

---

## ✨ Cover Image Feature (Highlight)

- Drag & drop or click to upload images  
- Instant preview after upload  
- Stored securely in cloud (EdgeStore)  
- Synced with database (Convex)  
- Real-time rendering in UI  
- Option to change or remove cover image  

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), React, Tailwind CSS  
- **Backend:** Convex (Real-time database)  
- **Authentication:** Clerk  
- **Storage:** EdgeStore  
- **UI Components:** Shadcn/UI
